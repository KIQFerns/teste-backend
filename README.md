## Description

Api com crud de usuário.

## Tecnology

- Node.js
- Express.js
- APIRestful
- Sequelize
- Postgress

## Installation

- **crie um arquivo **.env na raiz do projeto para inserir as informações do seu banco em desenvolvimento (tem um modelo na raiz: env.example);

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start
```
